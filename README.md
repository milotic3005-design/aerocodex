# Aerospace Engineering — A Free Self-Study Codex

A complete, self-directed aerospace engineering curriculum — from algebra to
orbital mechanics — built entirely from free and open-access resources.

Seven phases, ~2,000 study hours, 100% free. Single self-contained `index.html`
(no build step, no dependencies, no framework runtime).

## Run locally
Just open `index.html` in any web browser.

## Deploy
This is a static site. Any static host works (Vercel, Netlify, GitHub Pages).
On Vercel, no configuration is needed — it serves `index.html` automatically.
